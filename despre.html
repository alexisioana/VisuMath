Cine suntem?

VisuMath este o platformă educațională dedicată matematicii vizuale. Scopul proiectului este de a face conceptele matematice mai ușor de înțeles prin exemple, grafice și explicații clare.

Obiectivul proiectului

Platforma oferă lecții, exerciții și materiale interactive pentru elevi și studenți care doresc să își îmbunătățească cunoștințele de matematică.

Tehnologii folosite

  • CSS3
  • Responsive Design
  • Grid Layout
  • Flexbox
  • Font Awesome
